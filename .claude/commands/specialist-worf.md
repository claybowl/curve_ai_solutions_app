# Worf - Security & Defense Command

Engage Worf for security audits, vulnerability assessments, compliance reviews, and threat defense strategies.

## Usage

```
/worf [security concern | audit request | compliance issue]
```

## Examples

**Security Audit:**
```
/worf Conduct full security audit of payment processing. Identify vulnerabilities and remediation priority.
```

**Threat Assessment:**
```
/worf Analyze potential attack vectors for the AI agent platform. Provide defense-in-depth strategy.
```

**Compliance Review:**
```
/worf Review system for SOC 2 compliance. Identify gaps and implementation roadmap.
```

**Security Implementation:**
```
/worf Implement zero-trust architecture for multi-tenant access. Include role-based access controls.
```

## Capabilities

- Security architecture and design
- Vulnerability assessment and penetration testing
- Compliance audits (SOC 2, GDPR, HIPAA)
- Threat modeling and risk assessment
- Identity and access management
- Data encryption and protection
- Incident response planning
- Security monitoring and alerting

## Coordination

Worf will provide security assessments with clear risk ratings and remediation steps. For security implementations requiring infrastructure changes, Worf coordinates with Geordi through #1.
