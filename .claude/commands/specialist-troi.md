# Troi - User Experience & Psychology Command

Engage Troi for user research, UX analysis, pricing psychology, team wellness, and customer insights.

## Usage

```
/troi [UX research | user analysis | psychological impact | team wellness]
```

## Examples

**User Experience Analysis:**
```
/troi Analyze user journey from signup to first AI agent deployment. Identify friction points and emotional blockers.
```

**Pricing Psychology:**
```
/troi Review the Donjon AiGency Suite pricing. Analyze psychological impact of tier differences and recommend optimizations.
```

**User Research:**
```
/troi Design user research study for AI agent adoption. Include interview questions and success metrics.
```

**Team Wellness:**
```
/troi Assess team workload and stress levels during product launch. Recommend wellness strategies and communication improvements.
```

## Capabilities

- User experience research and analysis
- Customer psychology and behavior insights
- Pricing psychology and optimization
- User journey mapping
- A/B testing design and interpretation
- Team wellness and morale assessment
- Communication strategy
- Accessibility and inclusion analysis

## Coordination

Troi will provide user-centric insights with actionable recommendations. For implementing UX changes, Troi coordinates with Geordi through #1.
