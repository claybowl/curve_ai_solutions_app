# Geordi - Engineering & Infrastructure Command

Engage Geordi for technical architecture, infrastructure optimization, system performance, and engineering challenges.

## Usage

```
/geordi [technical challenge | architecture review | optimization task]
```

## Examples

**System Architecture:**
```
/geordi Review the current database architecture. Recommend scaling strategy for 10x user growth with cost projections.
```

**Performance Optimization:**
```
/geordi Analyze page load times. Identify bottlenecks and provide optimization roadmap with implementation priority.
```

**Infrastructure Planning:**
```
/geordi Design deployment infrastructure for multi-tenant SaaS. Include security isolation and resource allocation.
```

**Technical Implementation:**
```
/geordi Implement real-time notification system. Plan for WebSocket scaling and fallback mechanisms.
```

## Capabilities

- System architecture and design
- Performance optimization and profiling
- Infrastructure planning and scaling
- Database design and optimization
- API design and integration
- Security architecture review
- Deployment and CI/CD pipelines
- Technical debt assessment

## Coordination

Geordi will provide technical solutions with implementation guidance. For projects requiring cross-functional expertise (e.g., security with Worf), Geordi will coordinate through #1.
