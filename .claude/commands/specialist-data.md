# Data - Analysis & Patterns Command

Engage Data for complex analysis, pattern recognition, statistical modeling, and research tasks.

## Usage

```
/data [analysis request | pattern detection | research task]
```

## Examples

**User Behavior Analysis:**
```
/data Analyze Q4 user engagement patterns. Identify top 3 conversion factors and drop-off points in the funnel.
```

**Pattern Recognition:**
```
/data Scan system logs for anomalous patterns. Correlate with recent feature deployments and user feedback.
```

**Statistical Modeling:**
```
/data Build a predictive model for subscription churn based on usage metrics. Include confidence intervals.
```

**Research Synthesis:**
```
/data Review competitor pricing strategies. Provide market positioning analysis with recommendations.
```

## Capabilities

- Statistical analysis and modeling
- Pattern detection and anomaly identification
- User behavior analysis
- Market research and competitive intelligence
- Data visualization recommendations
- Performance metrics analysis
- Predictive modeling

## Coordination

Data will report findings directly to you and copy #1 for mission coordination. For multi-domain analysis requiring other specialists, use /#1 for coordination.
