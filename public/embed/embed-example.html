Clean Machine Chat Widget

This page demonstrates how to embed the Clean Machine Virtual Concierge chat widget on any website.

How to Embed This Chat Widget

To add this chat widget to your website, simply add the following script tag to your HTML, right before the closing </body> tag:

<script src="https://cleanmachinemobile.com/embed/chat-embed.js"></script>

That's it! The chat widget will automatically appear in the bottom-right corner of your website.

Important Notes

  • The chat widget is designed to be responsive and will work on both desktop and mobile devices.
  • All chat conversations are processed through our secure API.
  • For any technical issues, please contact our support team.
